# Shapes — 구현 및 수정 이력

> Godot 4.3 / C# 모바일 2D 아케이드 슈터  
> 뷰포트: 720×1280 (세로)  
> 메인 씬: `scenes/main/Game.tscn`

---

## 1. 현재 구현 스펙

### 1.1 Autoload

| 이름 | 역할 |
|------|------|
| `GameManager` | 점수, 게임 시간, `IsPaused`, `IsGameOver`, `IsBossActive`, `IsAbilitySelectActive` |
| `InputSettings` | 조이스틱 / 화면 드래그 조작 방식 저장 |
| `ThemeSettings` | 다크 / 라이트 테마 (기본값: **라이트**) |

일시정지는 `GetTree().Paused`를 쓰지 않고 **`IsPaused` 소프트 일시정지**로 처리한다.  
게임 오브젝트가 각자 `IsPaused`를 확인해 이동·발사·스폰을 멈춘다.

### 1.2 조작

- `TouchInput` (`Game.tscn` → `TouchLayer/TouchInput`) — 전체 화면 `Control`
- **조이스틱**: 링·노브 표시
- **화면 드래그**: 첫 터치 위치 마커 표시
- 최초 실행 시 조작 방식 선택 오버레이 (`ControlSetupOverlay`)
- `MobileInput` — 플레이어 이동 벡터 제공

---

### 1.3 플레이어 (`Player.cs`, `Player.tscn`)

- 원형 본체, 자동 직선 상향 발사
- **기본 스탯**: 초당 1발, 이동속도 330px/s, 탄환 속도 330px/s
- 레벨업 시 스탯 증가:
  - 이동속도 **+5** / 레벨
  - 발사 간격 -0.025초 / 레벨 (최소 0.12초)
  - 탄환 데미지 +1 / 레벨
- 체력 5, 피격 시 1초 무적 + 깜빡임
- EXP·레벨·체력·스탯 시그널 → HUD 연동
- 아이템 버프 적용·만료 처리
- 발사체 속도 = `기본 × 아이템 배율 × 능력 배율`

---

### 1.4 적 (`Enemy.cs`, `Spawner.cs`)

- 화면 상단 스폰, 직선 하강
- 초기 속도 90px/s, **6초마다 +2**
- **스폰 간격** (경과 시간 기준):

| 구간 | 간격 |
|------|------|
| 0~25초 | 3.50초 (웜업) |
| 25~60초 | 3.50 → 2.87초 (선형) |
| 60~120초 | 2.87초 |
| 120~180초 | 2.34초 |
| 180~240초 | 1.99초 |
| 240~300초 | 1.78초 |
| 300초~ | 1.50초 |

- 체력: `1 + 게임시간 × 0.075` (스폰 시 결정, 오각형은 1.5~2배 배율)
- **사각형**: 40px, EXP 5
- **오각형**: 18% 확률, 56px, EXP 1.5배
- **보너스 EXP**: 처치 시 5% 확률로 3배 (`RollExpReward`)
- 보스전 중 일반 적 스폰 중단

---

### 1.5 레벨·경험치

- 레벨당 필요 EXP: `18 × 현재 레벨`
- 레벨업 HP 보너스 없음
- **3레벨마다** 능력 선택 (3개 중 1개)

### 1.6 능력 시스템

#### 관련 파일

| 파일 | 역할 |
|------|------|
| `AbilityKind.cs` | 능력 종류 enum |
| `AbilityRarity.cs` | 등급 enum (레어/에픽/유니크/히든) |
| `AbilityChoice.cs` | 선택지 데이터 (`Title`, `Description` 등) |
| `AbilityRoller.cs` | 등급·종류 롤, **능력 설명 문구 정의** (플레이어 보유 능력 조건 반영) |
| `PlayerAbilities.cs` | 플레이어 보유 능력 누적 |
| `AbilityIcon.cs` | 선택 UI 아이콘 |
| `HUD.cs` | 능력 선택 오버레이 표시 |

#### 능력 종류

| 능력 | 설명 |
|------|------|
| 직진형 발사체 | 본체 양옆 동료, 직선 탄환 (공격력 본체의 50%, **초당 0.6발 고정**) |
| 사선형 발사체 | 본체 양옆 동료, ±30° 사선 탄환 (공격력 본체의 50%, **초당 0.6발 고정**) |
| 유도탄 | 본체 양옆 동료, 유도탄 (공격력 본체의 **135%**, **초당 0.4발 고정**, 에픽 이상) |
| 공격 강화 | 공격력·공격속도 % 증가 (레벨업 보너스에도 적용) |
| **발사체 속도 증가** | 등급별 발사체 속도 % 증가 (영구 누적) |
| **발사체 튕기기** | 직진·사선 탄환이 좌우 벽에 반사 (사선탄 보유 시 등장, 획득마다 +1회) |
| **발사체 관통** | 직진·사선 탄환이 적·보스 관통 (직진/사선 보유 시 등장, 획득마다 +1회) |

#### 발사체 속도 증가 (능력) 등급별

| 등급 | 증가량 |
|------|--------|
| 레어 | +10% |
| 에픽 | +30% |
| 유니크 | +50% |
| 히든 | +100% |

#### 등급별 확률

- 히든 1% / 유니크 5% / 에픽 13% / 레어 81%

#### 등급별 발사체 수 (직진·사선)

- 레어 1 / 에픽 2 / 유니크 3 / 히든 5

#### 능력 등장 조건

| 능력 | 조건 |
|------|------|
| 유도탄 | 에픽 이상 등급 롤 시 풀에 포함 |
| 발사체 튕기기 | **사선탄 1개 이상** 보유 |
| 발사체 관통 | **직진 또는 사선탄 1개 이상** 보유 |

### 1.7 동료 발사체 (`Companion.cs`, `Companion.tscn`)

- 능력 획득 시 **작은 동료 엔티티**가 본체 양옆에 생성
- 동료 종류별 색상 구분 (직진=파랑, 사선=초록, 유도=보라)
- 동료 반지름: 11px (본체 20px)
- 동료 위치: 본체 좌·우, 가장자리 사이 8px 간격
- 직진·사선: 동료마다 독립 발사, **초당 0.6발 고정** (본체 공속과 무관)
- **본체·직진·사선·유도탄** 모두 **화면 경계(상단·좌우) 도달 시 소멸** (3초 수명 제거)
- 유도탄: **초당 0.4발 고정**, 공격력 본체의 135%, 탄환 속도 420px/s
- **튕기기·관통**은 직진·사선·본체 탄환에 적용 (유도탄 제외)
- 피격 무적 시 본체와 함께 깜빡임

---

### 1.8 보스 (`Boss.cs`, `BossSpawner.cs`)

- 3분 / 6분 / 9분에 등장 (180초 간격)
- 체력: 1000 / 3200 / 5500
- 등장 시 일반 적 제거
- 하강 후 좌우 이동 + 플레이어 조준 탄막
- 처치 시 점수·EXP 지급 + **능력 3개 중 1개 선택** (보스 보상)
- 처치 시 **HP 회복 아이템 확정 드롭**

---

### 1.9 테마

- **다크**: 어두운 배경, 밝은 UI
- **라이트**: 베이지 배경, 어두운 UI (기본값)
- 색상 정의: `ThemePalette.cs`
- 설정 → 일시정지 메뉴에서 전환

### 1.10 UI / HUD (`HUD.cs`, `HUD.tscn`)

- 좌측 상단: 점수, HP, 공격력, 공격속도, 이동속도 (세로 나열)
- 좌측 스탯 영역: **반투명 배경 패널** (다크·라이트 공통, 투명도 28%)
- 우측 상단: 일시정지 버튼 (52×52 정사각형)
- 중앙: 타이머
- 하단: 레벨, EXP 바
- 오버레이: 조작 설정, 일시정지, 능력 선택, 게임 오버
- 능력 선택 중에는 일시정지 메뉴 숨김
- 일시정지 메뉴 재개 문구: 「이어서」
- **개발용** 우측 하단 `LV+1` 버튼 (`DevLevelUpButtonEnabled`, 출시 전 비활성화)

### 1.11 아이템 드롭 (`ItemPickup.cs`, `ItemDropper.cs`)

- **버프 아이템**: 적·보스 처치 시 **8%** 확률 (6종 중 1개)
- **HP 회복**: 적 처치 시 **5%** / **보스 처치 시 100%**
- 드롭 위치에 **고정**, 아이콘으로 표시
- 플레이어 접촉 시 획득
- 본체에 **약한 자석 효과** (반경 96px, 최대 끌림 200px/s)
- 같은 아이템 재획득 시 지속 시간 갱신, 여러 버프는 곱연산 적용

| 아이템 | 효과 | 지속 |
|--------|------|------|
| 데미지 ↑ (빨강) | 공격력 +50% | 5초 |
| 데미지 ↑ (주황) | 공격력 +30% | 10초 |
| 공속 ↑ (노랑) | 공격속도 +300% | 5초 |
| 공속 ↑ (연노랑) | 공격속도 +200% | 5초 |
| 이동 ↑ (초록) | 이동속도 +30% | 10초 |
| 발사체 속도 ↑ (하늘) | 발사체 속도 ×2, 공격속도 +15% | 10초 |
| HP 회복 (분홍) | HP +1 (풀체력이면 효과 없음) | 즉시 |
