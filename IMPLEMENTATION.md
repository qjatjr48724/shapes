# Shapes — 구현 및 수정 이력

> Godot 4.3 / C# 모바일 2D 아케이드 슈터  
> 뷰포트: 720×1280 (세로)  
> 메인 씬: `scenes/main/Game.tscn`

---

## 1. 초기 구현 내용

### 1.1 Autoload

| 이름 | 역할 |
|------|------|
| `GameManager` | 점수, 게임 시간, `IsPaused`, `IsGameOver`, `IsBossActive`, `IsAbilitySelectActive` |
| `InputSettings` | 조이스틱 / 화면 드래그 조작 방식 저장 |
| `ThemeSettings` | 다크 / 라이트 테마 (기본값: **라이트**) |

일시정지는 `GetTree().Paused`를 쓰지 않고 **`IsPaused` 소프트 일시정지**로 처리한다.  
게임 오브젝트가 각자 `IsPaused`를 확인해 이동·발사·스폰을 멈춘다.

---

### 1.2 조작

- `TouchInput` (`Game.tscn` → `TouchLayer/TouchInput`) — 전체 화면 `Control`
- **조이스틱**: 링·노브 표시
- **화면 드래그**: 첫 터치 위치 마커 표시
- 최초 실행 시 조작 방식 선택 오버레이 (`ControlSetupOverlay`)
- `MobileInput` — 플레이어 이동 벡터 제공

---

### 1.3 플레이어 (`Player.cs`, `Player.tscn`)

- 원형 본체, 자동 직선 상향 발사
- 레벨업 시 스탯 증가:
  - 이동속도 +20 / 레벨
  - 발사 간격 -0.025초 / 레벨 (최소 0.12초)
  - 탄환 데미지 +1 / 레벨
- 체력 5, 피격 시 1초 무적 + 깜빡임
- EXP·레벨·체력 시그널 → HUD 연동

---

### 1.4 적 (`Enemy.cs`, `Spawner.cs`)

- 화면 상단 스폰, 직선 하강
- 초기 속도 90px/s, 6초마다 +2
- 스폰: 초반 25초간 3.5초 간격 → 이후 점점 단축 (최소 0.65초)
- 체력: `1 + 게임시간 × 0.15` (오각형은 1.5~2배 배율)
- **사각형**: 40px, EXP 5
- **오각형**: 18% 확률, 56px, EXP 1.5배
- **보너스 EXP**: 처치 시 확률적으로 배율 적용 (`RollExpReward`)

---

### 1.5 레벨·경험치

- 레벨당 필요 EXP: `18 × 현재 레벨`
- 레벨업 HP 보너스 없음
- **5레벨마다** 능력 선택 (3개 중 1개)

---

### 1.6 능력 시스템

#### 관련 파일

| 파일 | 역할 |
|------|------|
| `AbilityKind.cs` | 능력 종류 enum |
| `AbilityRarity.cs` | 등급 enum (레어/에픽/유니크/히든) |
| `AbilityChoice.cs` | 선택지 데이터 (`Title`, `Description` 등) |
| `AbilityRoller.cs` | 등급·종류 롤, **능력 설명 문구 정의** |
| `PlayerAbilities.cs` | 플레이어 보유 능력 누적 |
| `AbilityIcon.cs` | 선택 UI 아이콘 |
| `HUD.cs` | 능력 선택 오버레이 표시 |

#### 능력 종류

| 능력 | 설명 |
|------|------|
| 직진형 발사체 | 본체 양옆 동료, 직선 탄환 (공격력·공속 본체의 50%) |
| 사선형 발사체 | 본체 양옆 동료, ±28° 사선 탄환 (공격력·공속 본체의 50%) |
| 유도탄 | 본체 양옆 동료, 유도탄 (공격력 본체의 33%, **초당 0.4발 고정**, 에픽 이상) |
| 공격 강화 | 공격력·공격속도 % 증가 (레벨업 보너스에도 적용) |

#### 등급별 확률

- 히든 1% / 유니크 5% / 에픽 13% / 레어 81%

#### 등급별 발사체 수 (직진·사선)

- 레어 1 / 에픽 2 / 유니크 3 / 히든 5

---

### 1.7 동료 발사체 (`Companion.cs`, `Companion.tscn`)

- 능력 획득 시 **작은 동료 엔티티**가 본체 양옆에 생성
- 동료 종류별 색상 구분 (직진=파랑, 사선=초록, 유도=보라)
- 동료 반지름: 11px (본체 20px)
- 동료 위치: 본체 좌·우, 가장자리 사이 8px 간격
  - 1명: 왼쪽
  - 2명: 왼쪽 + 오른쪽
  - 3명 이상: 양옆에서 바깥으로 확장
- 직진·사선: 동료마다 독립 발사 타이머 (본체 공속의 50%)
- 유도탄: 초당 0.4발 고정 (본체 공속과 무관)
- 피격 무적 시 본체와 함께 깜빡임

---

### 1.8 보스 (`Boss.cs`, `BossSpawner.cs`)

- 3분 / 6분 / 9분에 등장 (180초 간격)
- 체력: 1000 / 3200 / 5500
- 등장 시 일반 적 제거
- 하강 후 좌우 이동 + 플레이어 조준 탄막
- 처치 시 점수·EXP 지급 + **능력 3개 중 1개 선택** (보스 보상)

---

### 1.9 테마

- **다크**: 어두운 배경, 밝은 UI
- **라이트**: 베이지 배경, 어두운 UI (기본값)
- 색상 정의: `ThemePalette.cs`
- 설정 → 일시정지 메뉴에서 전환

---

### 1.10 UI / HUD (`HUD.cs`, `HUD.tscn`)

- 상단: 점수, 체력, 일시정지 버튼
- 중앙: 타이머
- 하단: 레벨, EXP 바
- 오버레이: 조작 설정, 일시정지, 능력 선택, 게임 오버
- 능력 선택 중에는 일시정지 메뉴 숨김 (`IsAbilitySelectActive`)

### 2026-06-20 — 초기 구현·안정화

1. Android에서 검은 화면이나 멈춤이 발생하던 문제를, 일시정지 중에도 화면과 입력이 유지되도록 수정했습니다.

2. 모바일에서 터치 조작이 동작하지 않던 문제를, 터치 입력이 화면 전체에서 잡히도록 수정했습니다.

3. 조작 방식 설정이 저장되지 않던 문제를 수정했습니다.

4. 5레벨 능력 선택 시 일시정지 메뉴가 함께 보이던 문제를, 능력 선택 중에는 일시정지 화면이 뜨지 않도록 수정했습니다.

5. 5레벨마다 능력을 선택하는 기능을 구현했습니다. 세 가지 후보 중 하나를 고릅니다.

6. 다크·라이트 테마를 선택할 수 있는 기능을 추가했습니다. 기본 테마는 라이트입니다.

---

### 2026-06-20 — 추가 수정

7. 동반 발사체가 본체 가운데에서만 퍼져 나가던 것을, 본체 양옆에서 본체와 같은 방향으로 쏘도록 수정했습니다.

8. 보너스 경험치가 나올 확률을 3%에서 5%로 올렸습니다.

9. 능력을 얻을 때 탄환만 추가되던 것을, 본체 양옆에 작은 동료가 생겨 발사하도록 수정했습니다.

10. 능력 선택 후 적과 탄환이 사라지던 문제를, 일시정지 중에는 그 자리에 남고 움직임만 멈추도록 수정했습니다.

11. 보스 처치 시 능력을 선택할 수 있는 보상을 추가했습니다. 레벨업 보상과 겹치면 순서대로 선택합니다.

12. 동료 발사체의 공격 속도가 본체의 50%가 되도록 수정했습니다. 동료마다 따로 발사하도록 바꿨습니다.

13. 보스 보상을 능력 1개에서 랜덤 3개 중 하나를 고르는 방식으로 변경했습니다.

14. 본체와 동료가 너무 붙어 보이던 문제를, 사이에 여유 간격을 두도록 수정했습니다.

15. 일시정지 메뉴의 「다시 시작」 문구를 「이어서」로 바꾸고, 상단 일시정지 버튼 높이를 줄였습니다.

16. 유도탄 공격 속도를 본체의 50%가 아닌 초당 0.4발로 고정했습니다.

## 3. 밸런스 상수 요약 (`GameConstants.cs`)

| 항목 | 값 |
|------|-----|
| 플레이어 체력 | 5 |
| 플레이어 발사 간격 (기본) | 0.35초 |
| 동료 공격력 비율 (직진·사선) | 50% |
| 동료 공격속도 비율 (직진·사선) | 50% |
| 유도탄 공격력 비율 | 33% |
| 유도탄 공격속도 | 초당 0.4발 (2.5초 간격) |
| 보너스 EXP 확률 | 5% (설계값) |
| 보너스 EXP 배율 | 3배 (설계값) |
| 레벨당 필요 EXP | 18 × 레벨 |
| 능력 선택 레벨 | 5, 10, 15… |
| 보스 등장 | 3분, 6분, 9분 |

> **참고:** `GameConstants.cs`에 테스트용으로 `BonusExpChance = 1f`, `BonusExpMultiplier = 5`가 주석과 함께 남아 있을 수 있다. 배포 전 설계값(5%, 3배)으로 되돌릴 것.
